Лабораторна робота №13

Використання функцій JavaScript

Виконавець: Врублевська Катерина, ПП-22

Мета роботи: навчитися використовувати стандартні і створювати функції користувача в JavaScript.

Виконані завдання:

  • Завдання 1. Вбудовані функції: Стилізоване відображення днів тижня (За вимогою)
  • Завдання 2. Вбудовані функції (Date): Розрахунок днів до Нового року
  • Завдання 3. Користувацька функція (без повернення): Динамічне оновлення ID-карти
  • Завдання 4. Користувацька функція (з поверненням): Генерація безпечного пароля
  • Завдання 5. Функція з необов'язковими/типовими параметрами: Конфігурація системи сповіщень
  • Завдання 6. Функція як аргумент (Callback): Фільтрація списку товарів за критерієм
  • Завдання 7. Функція як змінна (Function Expression): Перевірка рядка на паліндром
  • Завдання 8. Масив arguments: Форматування довільної кількості тегів/міток
